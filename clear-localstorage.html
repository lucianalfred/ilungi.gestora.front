Apagar dados locais do Gestora

Este utilitário remove as chaves do localStorage relacionadas à aplicação.



  

  

  
Ou cole este snippet na consola do navegador (mais rápido):

  
const keys = Object.keys(localStorage).filter(k => k.startsWith('gestora_') || k.includes('gestora')); keys.forEach(k => localStorage.removeItem(k)); console.log('Removidas', keys.length, 'chaves');
